Detailed Description

template<size_t bit_width>
class Synchrotron::SynchrotronComponent< bit_width >

SynchrotronComponent is the base for all components, offering in and output connections to other SynchrotronComponent.

Parameters
bit_widthThis template argument specifies the width of the internal bitset state.
